### 사용자 유형별 전시회 추천 웹페이지 
____________________________________________________________
한서대학교 항공소프트웨어공학과 데이터베이스
<br>
개발기간 : 2022. 11 ~ 2022. 11

## 프로젝트 소개

테스트를 통한 각 유저의 전시회 유형을 파악하고 해당 전시회를 추천하는 웹페이지


## 시작 가이드

### Requirements
For building and running the application you need:

- Python 3.10.5
- Django
- mysql


### Installation

    pip install pillow
    pip install argon
